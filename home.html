Linked-List-Addition (2023)

An addition calculator I made using Java utilizing linked lists to make it possible to add big integers. It puts the numbers in a linked list backwards then start adding the two integers from the 2 linked lists from left to right. The program then checks if the sum of the two integer from the current node is bigger than 9, if it is then we take the modulo of it per 10 integers. Then adds 1 to the value of the next two integers that would be added.
